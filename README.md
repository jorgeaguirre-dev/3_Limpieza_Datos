## Análisis Geográfico de Ataques
![Mapa Global de Ataques de Tiburón](./mapa_ataques_tiburones.png)

En este mapa se muestra la distribución global de ataques de tiburón, con cada país coloreado según el número total de ataques registrados. Los países con más ataques aparecen en tonos más oscuros de rojo, lo que permite identificar rápidamente las regiones más afectadas por este fenómeno.
